# TUI Overview

The `@comma-agents/tui` package provides a terminal-based user interface built with [Ink](https://github.com/vadimdemedes/ink) (React for the terminal). It connects to the daemon over WebSocket to run and observe agent strategies.

## Quick Start

```bash
# From the repo root
bun run tui
```
