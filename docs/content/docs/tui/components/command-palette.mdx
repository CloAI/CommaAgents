# CommandPalette

A modal command palette with a search input and scrollable command list. Each
command opens a focused sub-page. Open it with Ctrl+P.

```tsx
import { CommandPalette } from "@comma-agents/tui";

<CommandPalette
  isVisible={isOpen}
  onClose={close}
  commands={customCommands}
/>
```

## CommandPaletteProps

| Prop | Type | Default | Description |
|---|---|---|---|
| `isVisible` | `boolean` | -- | Whether the palette is visible |
| `id` | `string` | -- | Stable Ink focus ID |
| `onClose` | `() => void` | -- | Dismiss callback |
| `commands` | `readonly Command[]` | Built-in commands | Custom command list |
| `initialCommandId` | `string?` | -- | Page to open immediately |

## Command

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Unique command identifier |
| `label` | `string` | Display name |
| `description` | `string` | One-line description |
| `keywords` | `readonly string[]?` | Extra fuzzy-match keywords |
| `page` | `React.ComponentType?` | Sub-page component (page command) |
Selecting a command mounts its page inside the palette.

## Built-in Commands

| Command | Type | Description |
|---|---|---|
| `help` | Page | Keyboard shortcut reference |
| `list-providers` | Page | Available model providers |
| `mcp-servers` | Page | Shared defaults and current-run MCP overrides |
| `run-picker` | Page | Browse and load persisted runs |
| `settings` | Page | User configuration |

## Navigation

- **Type to filter** -- fuzzy-matches against label, description, and keywords
- **Up/Down arrows** -- move selection
- **Enter** -- activate selected command
- **Esc** -- pop sub-page back to home; close palette at home

The MCP page uses Intro scope to edit shared defaults and Chat scope to edit
the current run. Press `g` in Chat scope to copy the highlighted run choice
back to its shared default. The status-bar MCP count is clickable and opens
this page directly.

The palette is typically rendered inside a `<Modal>` for backdrop dimming.
