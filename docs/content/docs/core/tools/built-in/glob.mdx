---
title: glob
description: Find files by glob pattern matching.
---

The glob tool finds files matching a glob pattern (e.g., `**/*.ts`). Results are sorted alphabetically and capped at a configurable maximum.

```ts
import { createAgent } from "@comma-agents/core";

const agent = createAgent({
  name: "explorer",
  model: "openai/gpt-4o",
  tools: ["glob"],
});
```

## Parameters

The agent provides these parameters when invoking the tool:

| Parameter | Type | Description |
|---|---|---|
| `pattern` | `string` | Glob pattern to match files against (e.g., `**/*.ts`, `src/**/*.test.ts`). |
| `path` | `string?` | Directory to search in. Defaults to the current working directory. |

## Behavior

- Returns matching file paths, one per line, sorted alphabetically.
- Results are capped at the default maximum (100). A footer message indicates when results are capped.
- Only matches files, not directories.
