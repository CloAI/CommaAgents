# Hooks

Hooks allow you to intercept and transform agent and tool behavior at various lifecycle points. There are two kinds of hooks: side-effect hooks (observe without modifying) and transform hooks (modify values in the pipeline).

Hooks are attached to agents **after creation** using `hookIntoAgent`:

```ts
import { createAgent, hookIntoAgent } from "@comma-agents/core";

const agent = createAgent({ name: "demo", model: "openai/gpt-4o" });

hookIntoAgent(agent, {
  beforeCall: [async (message) => console.log("Calling:", message)],
  afterCall: [async (response) => console.log("Response:", response)],
});
```

Multiple `hookIntoAgent` calls stack — each call appends to the existing hooks.
